# Math Prerequisites
**Canonical location for math_prerequisites.md content**

## Required
- Linear algebra: matrix multiply, outer product, eigenvalues, SVD
- Calculus: chain rule, gradient, Jacobian
- Probability: expectation, variance, Gaussian
- Optimization: gradient descent, convexity, strong convexity
- Basic neural networks: MLP, backprop, Adam

## Helpful but Not Required
- Online learning / regret analysis (reviewed in Appendix A)
- Bilevel optimization (introduced in Chapter 3)
- Kernel methods (linear attention uses kernel ideas)

## Not Required
- PAC learning, VC dimension
- Statistical learning theory (Rademacher complexity)
- Information theory beyond mutual information intuition
