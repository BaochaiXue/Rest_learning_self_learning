# 第 9 章：Titans 与多时间尺度记忆系统

## 本章想回答什么问题？
当你遇到一个能够更新自身内部 Hidden Layers 参数的新型流组件（如 TTT Layers）时，你会发现一个致命缺陷：时间长度长了以后，由于它不断在同一空间 $S_t$ 进行更新和重置，**早期的细节很容易被不可逆地遗忘！** 怎么才能把短期的注意力敏锐捕获（Attention）与可以永远持续存续并能随时翻阅长期存档记忆体系结合起来？

## 混合巨物：Titans 架构
如果我们将不同更新手段和响应周期分拆，大脑可能不是统一处理的。我们建立一种模型：
- **第一级：Short-term Memory (基于传统自注意力提取流)**。负责捕获最近几百到几千字的关键句法语境逻辑和直接因果联系。它精准度高并可以进行直接高并发查询。
- **第二级：Neural Memory (长期神经网络挂载记忆)**。这不再是存在 KV Cache 里面的过去矩阵，而是类似于我们在之前讲过的模型“附带的那个独立的前馈神经结构块（例如包含可被参数化的自上而下矩阵或者深层组合）”。当新的序列特征产生并与当前记忆库出现极度偏离（不合预期）时（**Surprise 惊讶度法则**），模型驱动梯度强制将这个新事件编入它的底层连接。

## 学习如何在测试时“记住”
在推断长文本序列的时候，这块 Neural Memory 的深层权重是**实质上会被根据上溯惊讶法则不断更新的**！由于它是带有深层次神经网络本身特征的挂载库，它兼取了压缩能力大、具备一定的内在世界重组与关联等属性。
Titans 通过这种宏伟的并联系统，完美化解了纯 RNN 类型结构对于精细任务丢失长距离信息的缺陷。这说明对于深层架构设计而言：不同时序的微调模型混合是终局的必然产物之一。

## 核心主公式（Surprise 梯度重铸存储）
对 Neural Memory 参数库 $W_M$ 更新，通常采用如下形式去削减重构误差：
$$ \Delta W_M \propto - \nabla_{W_M} || M_{W_M}(k_t) - v_t ||^2 $$
模型在此过程中自行决定学习率与更新门限，动态控制写出和持久化操作的历史。

## 代表论文
- Behrouz, Ali, et al. *"Titans: Learning to Memorize at Test Time."* 2025. 通过长时记忆组件与短平快注意力模块联合作战，使得长上下文推演与搜索突破性能限界的巨作。

## 观点和机制的边界
我们把这也列在教课书中。但由于目前依然属于前沿探勘阶段：Titans 在这种“多脑工作协同机制”展现了威力，可大家不要将其神化为人类大统一多级海马体系统，它仍旧是一个具备极强启发意义与工程突破的混合模块尝试机制代表。

## 可做的 Lab
**长文本大海捞针 Toy Problem (Needle in A Haystack)!** 
使用 PyTorch 构建随机多级数字流，在 1 处种下 “密文”。使用单纯的 Attention + 限定窗长度产生必然记忆掉落。随后在它的流旁边并联加上一个小小的由一阶 MomentumSGD 更新的单层 Neural Memory 组件并混合输出，验证提取出的结果。

## 常见误解
- **误解**：Neural Memory 只是把所有的字压缩成了 $Z$ 存进了另外一个更大的向量池里。
- **纠正**：它其实是“将这些内容和他们的互相匹配特性消化在了一次次梯度写操作带来的网络权重改写中”。并不是原始池向量叠加，而是通过权重映射完成的信息拓扑融合。
