# Reproducibility Checklist

## Environment

- [ ] Runtime and OS captured
- [ ] Dependency versions captured
- [ ] Hardware assumptions recorded when relevant

## Data And Evaluation

- [ ] Data source and split logic recorded
- [ ] Evaluation protocol recorded
- [ ] Metrics defined unambiguously
- [ ] Leakage risks reviewed

## Randomness And Stability

- [ ] Seed strategy recorded
- [ ] Number of runs or restarts recorded
- [ ] Variance or confidence discussion recorded when relevant

## Artifacts

- [ ] Exact config or script path recorded
- [ ] Output artifact path recorded
- [ ] Findings log points to the relevant artifacts

- Risk level:
- Blocking items:
- Safe to share externally: yes / no
