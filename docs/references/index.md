# docs/references/index.md

This directory is for source inventories and canonical external anchors.

## Use It For

- Paper links
- Benchmark specs
- Official docs
- Repository URLs
- Versioned standards or schemas

## Conventions

- Prefer primary sources over summaries.
- Record the source type, access date, and why the source matters.
- Link out instead of copying large excerpts into docs.
- If a source is relevant to a claim, point the claim to the exact reference entry.
