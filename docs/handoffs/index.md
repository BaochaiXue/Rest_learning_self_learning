# docs/handoffs/index.md

This directory captures resumable checkpoints for the root research OS.
Use a handoff when a phase is complete, when work crosses agent boundaries, or when a future session should be able to resume without reconstructing context from chat.

## Files

- `latest_handoff.md`: the current handoff snapshot

## Handoff Rules

- Keep the handoff short and concrete.
- Include what changed, what evidence exists, what is still open, and what should happen next.
- Update it after substantial root-level work.
