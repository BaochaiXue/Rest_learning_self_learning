# docs/design-docs/index.md

This directory holds durable design beliefs for the root research operating system.
If a policy needs to stay true across many tasks, agents, and sessions, it belongs here.

## Contents

- `core-beliefs.md`: the operating beliefs that shape the rest of the docs tree

## When To Update

- Update this directory when a root-level workflow pattern becomes stable enough to reuse.
- Do not use it for topic-specific research notes.
- Do not use it for task logs or experimental results.
