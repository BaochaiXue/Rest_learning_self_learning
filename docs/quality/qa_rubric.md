# QA Rubric

Use this rubric to judge whether a root docs artifact is ready to rely on.

## Criteria

- Traceability: every non-trivial claim points to a source, artifact, or file path.
- Scope control: the doc says what it covers and what it does not cover.
- Freshness: the doc names what is current and what is stale.
- Cross-linking: related files are linked instead of re-explained.
- Reproducibility: readers can follow the same steps or checks again.
- Recovery: the doc says what to do if the work is blocked or incomplete.
- Readability: the doc is short enough that a future agent can scan it quickly.

## Practical Threshold

- Pass: all critical items are present and no major evidence gap remains hidden.
- Needs work: the artifact is useful but leaves important evidence or scope unclear.
- Fail: the artifact makes claims that cannot be traced or resynthesized.
